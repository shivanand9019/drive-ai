
# DriveAI

DriveAI is a secure cloud storage platform that allows users to upload, manage, share, and summarize documents using AI.

## Tech Stack

Backend:
- Java
- Spring Boot

Frontend:
- React

Database:
- PostgreSQL

Storage:
- MinIO

AI:
- Python
- FastAPI
- Gemini API

## Features

- Secure Login
- File Upload
- File Sharing
- AI Document Summary
- Dashboard
